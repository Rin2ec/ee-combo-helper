## 🛠 工具說明
**伊伊接技小幫手** 是一個用於練習鍵盤組合操作的工具，適用於各種需要鍵盤組合的遊戲。主要功能包括：
- 讀取指定格式的txt檔案，顯示按鍵組合。
- 置頂視窗、隱藏介面、更改按鍵語言或模式、提示功能、系統訊息等。
- 詳細可以查看[巴哈姆特文章](https://)。

### 🚀 使用方法
1. 下載並解壓縮或安裝本工具。
2. 將需要的txt檔案放置於對應目錄中。
3. 開啟工具並讀取檔案，隱藏UI後開始練習鍵盤組合操作。
